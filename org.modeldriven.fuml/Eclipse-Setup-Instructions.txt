1.) Download and install Java 8 JDK and set your JAVA_HOME
environment variable to the JDK root directory.

    See http://www.oracle.com/technetwork/java/javase/downloads/index.html

2.) Download and install Maven 2.2.1 (not 3)

    See http://maven.apache.org/download.cgi

3.) Download and install Ant

    See http://ant.apache.org/bindownload.cgi

4.) Download an Eclise version or derivitive with Maven 2 (M2) 
support. Recommend Spring Tools Suite (STS). 
    
    See http://www.springsource.org/sts

5.) Launch Eclipse (or derivitive) and import the
fUML project.

    File->Import->Maven->Existing Maven Projects

    (Browse to the project "parent" directory, select the project, Finish.) 

6.) Test the project

    (Right-Click the project root)->Run As->Maven Test
      
7.) Install the project

    (Right-Click the project root)->Run As->Maven Install
